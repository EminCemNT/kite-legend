## ✨ 游戏特色

### 🚀 雷电风格射击
- **自动射击系统**：风筝持续向前方发射能量波
- **武器升级**：LV1 单发 → LV2 三发散射 → LV3 五发弹幕
- **炸弹清屏**：按下 B 键释放全屏 EMP 冲击波

### 🎯 敌机系统
| 敌机 | 攻击方式 | 分数 |
|------|---------|------|
| 小型侦查机 | 单发点射 | 100 |
| 中型攻击机 | 扇形弹幕 | 300 |
| 重型堡垒 | 全向弹幕 | 500 |
| Boss 机甲 | 复杂弹幕 + 模式切换 | 2000 |

### 🌍 动态场景
- 🌿 **草原上空**（0-300m）— 飘落叶片、绿地背景
- ☁️ **云海之上**（300-800m）— 白云翻涌、蓝天渐变
- ⚡ **高空雷电**（800-1500m）— 暗云闪电、暗色调
- 🌌 **极光边界**（1500-2500m）— 动态极光、满天繁星
- 🛸 **浩瀚太空**（2500m+）— 深空星海、陨石雨

### 🔥 游戏机制
- **Combo 连击**：连续击杀显示连击数，分数倍增
- **Fever 狂热模式**：击杀积攒能量条，满格进入无敌极速射击
- **天气系统**：随机切换晴天 / 暴风雨 / 彩虹 / 流星雨
- **Boss 战**：每 500 米遭遇巨型机甲 Boss

### 💎 道具系统
🔴 红胶囊（武器升级） · 💣 蓝胶囊（炸弹+1） · ⭐ 星星（2倍分） · 🛡️ 护盾（无敌） · 🪙 金币 · ❤️ 生命

### 🎨 Game Juice 特效
屏幕震动 · 命中停顿 · 粒子爆炸 · 挤压拉伸 · 速度线 · 拖尾光粒子

| 操作 | PC | 手机 |
|------|-----|------|
| 移动风筝 | ← → ↑ ↓ 方向键 | 手指拖拽 |

## 🛠️ 技术栈

- **纯原生 HTML5 + JavaScript + Canvas**
- **零依赖**：单个 HTML 文件，打开即玩
- **响应式设计**：PC / 手机触屏双端适配
- **localStorage**：自动保存最高分和金币

## 🪁 星际风筝传说

融合中国传统风筝文化与街机射击爽感（灵感来源：经典街机《雷电》），操控科幻风格的战机风筝，飞越草原、云海、极光、太空，迎战各种敌机与 Boss！

**核心亮点**：
- 向上飞的竖向卷轴视角
- 武器升级 + 弹幕系统 + Boss 战
- 5 大动态主题场景
- 天气系统 + Fever 狂热模式
- 丰富的 Game Juice 手感反馈

**操作**：PC 用方向键移动，B 键释放炸弹；手机触屏拖拽。

一个文件，打开浏览器即玩！
